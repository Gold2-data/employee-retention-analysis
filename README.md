# Employee Retention Analysis

This project analyzes employee data to identify factors affecting employee retention and builds a logistic regression model to predict employee turnover.

## Objectives

- Perform data cleaning (null values and duplicates)
- Conduct exploratory data analysis
- Visualize salary and department impact on retention
- Build a logistic regression model
- Evaluate model accuracy

## Tools Used

- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Results

The logistic regression model achieved approximately 86% accuracy.
